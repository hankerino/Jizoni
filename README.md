# Jizoni

This project is a web application for managing construction and engineering projects.

## Running the application

To run the application, you need to have Node.js and npm installed on your machine.

1. Clone the repository: `git clone https://github.com/your-username/Jizoni.git`
2. Install the dependencies: `npm install`
3. Start the development server: `npm run dev`

The application will be available at `http://localhost:5173`.
